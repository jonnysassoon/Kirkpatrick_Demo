Kirkpatrick's Algorithm

I created an application in Processing that provides an interactive demo and visualization of Kirkpatrick's point location algorithm. (You can find the original paper here).

Instructions:

  1. If you do not have it installed already, install the Processing application. You can download it here.
  2. Next, clone/download my repository on github. Read the README
  3. Once the files are downloaded, simply open one of the files in Processing (this will open the entire folder), and run the program.
